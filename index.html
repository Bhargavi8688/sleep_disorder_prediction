About

This project is a web-based sleep disorder prediction system designed to help users assess their sleep health using personal and lifestyle data. The application collects details such as age, gender, blood pressure, heart rate, sleep duration, BMI category, snoring pattern, diet habits, stress-related behavior, previous health history, and work hours to generate a prediction report. It uses an ensemble learning approach that combines K-Nearest Neighbors (KNN), Support Vector Machine (SVM), and Random Forest models to classify the user's condition into Low, Moderate, or High risk levels. In addition to prediction, the system provides secure user registration and login, stores generated reports for later review, and includes an admin module for monitoring records and supporting doctor consultation workflow.
